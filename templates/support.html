Frequently Asked Questions

How do the AI alerts work?

Alerts are generated daily at market close. Our ROMA multi-agent framework fetches price data, scans Farcaster sentiment via Neynar, runs forecasting, and synthesizes it into actionable alerts.

How do I add more portfolios?

Navigate to the Portfolios page and use the "Quick Create" widget on the left side to add new asset collections instantly.

Why am I not seeing new sentiment data?

Make sure you have correctly set up your `NEYNAR_API_KEY` in the `.env` file of your application. The Farcaster search requires a valid API key.
